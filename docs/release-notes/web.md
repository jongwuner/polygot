# Web Client Release Notes

## v1.3.0 - 2026-03-14

- Added Obsidian path configuration from the web editor.
- Saved notes into `{base}/{language}/polygot/{YYYY-MM-DD}/...` when Obsidian settings exist.
- Kept markdown download as the fallback when an Obsidian vault is not configured.

## v1.2.0 - 2026-03-14

- Expanded quick insert coverage for Korean and German.
- Added broader Japanese romaji mappings.
- Improved Chinese tone editing and `v` handling for umlaut input.

## v1.1.0 - 2026-03-14

- Split the original single-file prototype into modular CSS and JavaScript files.
- Separated dictionaries, IME handlers, editor logic, and bootstrap code.
- Made future editor changes easier to isolate by concern.

## v1.0.0 - 2026-03-14

- Shipped the initial multilingual web editor.
- Added Korean, English, Japanese, Chinese, and German writing modes.
- Added keyboard shortcuts, copy, clear, save, and Firebase Hosting deployment.
